Description détaillée

Implémentation d'un système de fichiers virtuel.

Ce fichier contient l'implémentation d'un système de fichiers simple en mémoire. Il fournit les fonctionnalités suivantes:

  • Gestion des fichiers et répertoires
  • Gestion des permissions
  • Liens durs et symboliques
  • Persistance des données
