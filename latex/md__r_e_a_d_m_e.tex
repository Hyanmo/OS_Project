\begin{DoxyItemize}

\begin{DoxyItemize}
\item Commande \+: {\ttfamily ls \mbox{[}chemin\mbox{]}}

\item {\bfseries{Changer de répertoire}}
\begin{DoxyItemize}
\item Commande \+: {\ttfamily cd chemin}
\item Exemples \+:
\begin{DoxyItemize}
\item {\ttfamily cd documents} (accéder au sous-\/répertoire)

\item {\ttfamily cd /} (retourner à la racine)
\end{DoxyItemize}
\end{DoxyItemize}
\item {\bfseries{Copier un fichier}}

\end{DoxyItemize}
\item {\bfseries{Modifier les permissions}}
\begin{DoxyItemize}

\item Commande \+: {\ttfamily read nom\+\_\+fichier}
\item Exemple \+: {\ttfamily read test.\+txt}
\end{DoxyItemize}
\item {\bfseries{Écrire dans un fichier}}
\begin{DoxyItemize}
\item Commande \+: {\ttfamily write nom\+\_\+fichier contenu}

\item {\bfseries{Créer un lien dur}}

\item {\bfseries{Quitter le programme}}
\begin{DoxyItemize}
\item Commande \+: {\ttfamily exit}

\end{DoxyEnumerate}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Système de Permissions}{Système de Permissions}}\label{md__r_e_a_d_m_e_autotoc_md3}
Les permissions suivent le format UNIX standard avec trois chiffres \+:
\begin{DoxyItemize}
\item 7 (rwx) \+: Lecture, écriture et exécution
\item 6 (rw-\/) \+: Lecture et écriture
\item 5 (r-\/x) \+: Lecture et exécution
\item 4 (r--) \+: Lecture seule
\item 3 (-\/wx) \+: Écriture et exécution
\item 2 (-\/w-\/) \+: Écriture seule
\item 1 (--x) \+: Exécution seule
\item 0 (---) \+: Aucune permission
\end{DoxyItemize}

Permissions courantes \+:
\begin{DoxyItemize}
\item 755 \+: Propriétaire (rwx), Autres (r-\/x)
\item 644 \+: Propriétaire (rw-\/), Autres (r--)
\item 777 \+: Tous les droits pour tous 
\end{DoxyItemize}
