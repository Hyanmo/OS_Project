\doxysubsection{Description détaillée}
Interface du système de fichiers virtuel. 

Ce fichier définit l\textquotesingle{}interface publique du système de fichiers, y compris les structures de données et les fonctions disponibles. Le système implémente les fonctionnalités suivantes\+:
\begin{DoxyItemize}
\item Création et suppression de fichiers et répertoires
\item Gestion des permissions
\item Lecture et écriture de fichiers
\item Navigation dans l\textquotesingle{}arborescence
\item Liens durs et symboliques
\item Persistance des données
\end{DoxyItemize}

Structure représentant un nœud dans le système de fichiers. 

Cette structure est utilisée pour représenter à la fois les fichiers et les répertoires. Elle contient toutes les métadonnées nécessaires ainsi que les liens vers d\textquotesingle{}autres nœuds.
